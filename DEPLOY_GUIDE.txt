# 🚀 QUICK DEPLOYMENT GUIDE

## STEP 1: GitHub Upload
1. Go to https://github.com/new
2. Create repository name (e.g., `saini-bot-speed`)
3. Upload ALL 15 files from this folder
4. Click "Commit changes"

## STEP 2: Heroku Deploy

### Method A: One-Click (Easiest)
1. Go to your GitHub repo
2. Click the "Deploy to Heroku" button in README
3. Fill environment variables
4. Deploy!

### Method B: Manual (More Control)
```bash
# Install Heroku CLI first

# Login
heroku login

# Create app
heroku create your-bot-name

# IMPORTANT: Add buildpacks in this order
heroku buildpacks:add --index 1 heroku-community/apt
heroku buildpacks:add --index 2 heroku/python

# Set environment variables
heroku config:set API_ID=28389286
heroku config:set API_HASH=b88da5f4f338cca30f8ea5fb53cb083b
heroku config:set BOT_TOKEN=your_bot_token_here
heroku config:set OWNER=6334323103
heroku config:set CREDIT="Your Credit"
heroku config:set AUTH_USERS=6334323103,123456789
heroku config:set TOTAL_USERS=6334323103,123456789

# Deploy
git clone https://github.com/yourusername/saini-bot-speed.git
cd saini-bot-speed
git init
git add .
git commit -m "Initial deploy"
heroku git:remote -a your-bot-name
git push heroku main

# Start worker
heroku ps:scale worker=1

# Check logs
heroku logs --tail --ps worker
```

## STEP 3: Verify Speed
Send a video link to your bot and check:
- Download speed should be 5-10 MB/s (vs 300 KB/s before)
- Progress bar should update faster
- ETA should be much shorter

## 🔧 TROUBLESHOOTING

### Buildpack Error?
```bash
heroku buildpacks:clear
heroku buildpacks:add --index 1 heroku-community/apt
heroku buildpacks:add --index 2 heroku/python
```

### Slow Speed?
Check if aria2c is installed:
```bash
heroku run aria2c --version
```

Should show: `aria2 version 1.36.0` or higher

### Bot Not Starting?
```bash
heroku ps:restart worker
heroku logs --tail
```
